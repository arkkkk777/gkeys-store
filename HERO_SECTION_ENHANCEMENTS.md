# Hero Section Enhancements with React Bits-inspired Components

## Overview
Hero секция главного экрана была улучшена с использованием компонентов, вдохновленных React Bits, сохранив всю существующую логику работы карусели и большой обложки.

## Созданные компоненты

### 1. GlitchText (`src/components/ui/glitch-text.tsx`)
- Анимированный текст с эффектом глитча
- Используется для заголовка игры в HeroContent
- Эффект активируется при hover
- Настраиваемая интенсивность (low, medium, high)
- Использует accent color #b4ff00

### 2. Aurora (`src/components/ui/aurora.tsx`)
- Фоновый эффект Aurora с плавными градиентами
- Добавлен как overlay в HeroContent
- Создает динамическое свечение на фоне
- Настраиваемая интенсивность и цвет
- Использует accent color #b4ff00

### 3. ClickSpark (`src/components/ui/click-spark.tsx`)
- Эффект искр при клике на кнопки
- Обертывает кнопки Buy и Wishlist
- Создает частицы, разлетающиеся от точки клика
- Настраиваемое количество искр и цвет

## Использование

Компоненты автоматически интегрированы в Hero секцию. Никаких дополнительных действий не требуется.

### GlitchText
```tsx
<GlitchText intensity="medium" style={{ fontSize: '48px' }}>
  {game.title}
</GlitchText>
```

### Aurora
```tsx
<Aurora color="#b4ff00" intensity={0.15} />
```

### ClickSpark
```tsx
<ClickSpark onClick={handleClick} color="#b4ff00">
  <Icons.Cart /> Buy
</ClickSpark>
```

## Цветовая схема
